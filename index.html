Apollo

A simple Python script that utilizes Youtube-dl to download songs.

How to Use

Linux/MAC OSX

  1. Install Python. On MAC OSX, and most linux distros, Python 2 is pre-installed
  2. Install ffmpeg through your distro's package manager. For MAC OSX, use homebrew
  3. Download the zip and extract the contents to a folder, or use git clone https://github.com/DeltawolfIII/Euterpe.git
  4. type python euterpe.py in terminal. Or if you have Python 3, type python3 euterpe.py.

Windows

  1. Install Python
  2. Install ffmpeg
  3. Download the zip and extract the contents to a folder.
  4. type python euterpe.py into Administrator Command Prompt
